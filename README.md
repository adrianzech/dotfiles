- **Window Manager**: bspwm

- **Shell**: zsh
- **Terminal**: urxvt
- **Compositor**: compton
- **Menu**: rofi

## Install

Run this:

```sh
git clone https://github.com/adrianzech/dotfiles.git ~/.dotfiles
cd ~/.dotfiles && ./install
```

This will symlink the appropriate files in `.dotfiles` to your home directory.
